# Vue Personal - a simple lightweight mobile UI

## Install 
```
npm install v-personal font-awesome
```

## Usage
```
import  'font-awesome/css/font-awesome.min.css';
import "./../node_modules/v-personal/v-personal.css"
import Personal from "v-personal";
Vue.use(Personal)
```
